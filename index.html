PitchAnnotator.Net

This is a desktop application implemented using WPF that gives user the ability to annotate lines and regions in a given collection of images.

Source Code

Source code and binaries for this project will be made available soon.

Annotating Lines

You can annotate straight lines. Up to this point, we only wanted to annotate straight lines. So only annotations of straight lines is implemented. However, new features can be added easily. Application can be extended by adding support for annotations of curved lines.

Player Segments

Also you can brush subjects of interest. For our research purpose, we wanted to annotate player segments, so we used means of brushing over players to annotate them. As can be seen in the picture below, a tranparent red brush overlays player segments.
